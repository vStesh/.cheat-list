# Bash cheat list

## General

`ls -la` - list current directory with parameters

`pwd` - show current path directory

`mkdir <dirName>` - create new directory

`cd <dirName>` - change current directory to the `dirName`

`rm -rf <dirName>` - remove directory `dirName`


## Sudo
> Add `sudo` before command for run as `root admin user`

`sudo lsof -i -P -n` - show active ports list

`ps` - show process list

`kill <pid>` - kill process with id

`df -h` - show free space memory on the disk
